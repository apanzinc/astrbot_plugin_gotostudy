# 去学习！ (GotoStudy)

[![AstrBot](https://img.shields.io/badge/AstrBot-Plugin-blue)](https://github.com/Soulter/AstrBot)
[![License](https://img.shields.io/badge/license-AGPL--v3-green)](LICENSE)

一个有趣的 AstrBot 插件，当指定 QQ 用户发送消息时，自动提醒他们去学习！

## ✨ 功能特性

- 🎯 **目标用户监控** - 指定特定 QQ 号，当他们在群里发言时自动回复
- 👥 **群白名单** - 支持设置只在特定群生效，或所有群都生效
- 💬 **自定义回复** - 可自定义回复消息内容
- 🔧 **双配置方式** - 支持 WebUI 可视化配置和聊天命令配置

## 📦 安装

在 AstrBot WebUI 的插件市场中搜索 "去学习" 并安装，或通过 GitHub 仓库安装。

## ⚙️ 配置

### WebUI 配置
在插件设置页面配置目标 QQ 号、群白名单和回复消息。

### 命令配置
```
/gotostudy add <QQ号>       # 添加目标用户
/gotostudy remove <QQ号>    # 移除目标用户
/gotostudy addgroup <群号>  # 添加群白名单
/gotostudy setmsg <消息>    # 设置回复消息
/gotostudy list             # 查看配置
/gotostudy help             # 显示帮助
```
